Place character face PNG images here.

Folder Structure:
/assets/faces/male/     <- Place male character images here (e.g. "alex.png")
/assets/faces/female/   <- Place female character images here (e.g. "belle.png")
/assets/faces/other/    <- Place other images here

Example:
If a character is "Male" and faceId is "hero-01", 
the game will look for: "/assets/faces/male/hero-01.png"
