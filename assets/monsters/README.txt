Place monster PNG images here.

Naming Convention:
The filename must match the ID defined in `MONSTERS_CSV` (in src/data/gameData.ts) exactly.

Examples:
- If the CSV ID is "SLIME", place "SLIME.png" here.
- If the CSV ID is "GOBLIN", place "GOBLIN.png" here.
- If the CSV ID is "ORC", place "ORC.png" here.

Recommended size: 256x256 or larger (images are scaled to fit, pixel art style recommended).
If no file is found here, the game will use the internal fallback graphics.
