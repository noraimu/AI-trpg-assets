
Place dungeon texture images here to override the default generated graphics.
Supported format: .png (Transparency recommended for objects)
Recommended size: 256x256 or 512x512

Naming Convention (Must be lowercase):

- wall.png          (Dungeon Walls)
- door.png          (Door Frame & Door)
- door_locked.png   (Locked Door)
- stairs.png        (Stairs Up/Down)
- fountain.png      (Recovery Spring)
- chest_red.png     (Fixed Treasure Chest)
- chest_blue.png    (Random Treasure Chest)
- spinner.png       (Spinning Floor Tile)
- arrow_n.png       (Conveyor Belt: North)
- arrow_e.png       (Conveyor Belt: East)
- arrow_s.png       (Conveyor Belt: South)
- arrow_w.png       (Conveyor Belt: West)

If a file is missing, the game will automatically use the built-in SVG graphics.
